## Aim
To study FastAPI-based ML inference deployment and compare FastAPI with Flask.

## Objective
- Build and serve a trained ML model via REST API using FastAPI
- Compare FastAPI vs Flask on performance and features
- Generate interactive Swagger UI documentation automatically
- Implement input validation using Pydantic models

## Theory

FastAPI vs Flask Comparison:

Feature          | Flask         | FastAPI
Input validation | Manual        | Automatic (Pydantic)
Async support    | Limited       | Native (async/await)
Documentation    | None built-in | Auto Swagger UI + ReDoc
Performance      | 2000-3000 r/s | 5000-8000 r/s
Type safety      | Optional      | Enforced

FastAPI uses Pydantic models for automatic request validation, Uvicorn as ASGI server for async performance, and generates /docs (Swagger UI) and /redoc automatically.

                                                                    ## Code: main.py (FastAPI Application)

                                                                        from fastapi import FastAPI
                                                                            from pydantic import BaseModel, Field
                                                                                from typing import List
                                                                                    import pickle, numpy as np, time

                                                                                        with open(\"model.pkl\", \"rb\") as f:
                                                                                                model = pickle.load(f)

                                                                                                    CLASS_NAMES = [\"setosa\", \"versicolor\", \"virginica\"]
                                                                                                    
                                                                                                        app = FastAPI(
                                                                                                                title=\"Iris Classification API\",
                                                                                                                        description=\"High-performance ML inference API\",
                                                                                                                                version=\"1.0.0\"
                                                                                                                                    )
                                                                                                                                    
                                                                                                                                        class IrisInput(BaseModel):
                                                                                                                                                sepal_length: float = Field(..., gt=0, le=20, example=5.1)
                                                                                                                                                        sepal_width:  float = Field(..., gt=0, le=20, example=3.5)
                                                                                                                                                                petal_length: float = Field(..., gt=0, le=20, example=1.4)
                                                                                                                                                                        petal_width:  float = Field(..., gt=0, le=20, example=0.2)
                                                                                                                                                                        
                                                                                                                                                                            class BatchInput(BaseModel):
                                                                                                                                                                                    instances: List[IrisInput]
                                                                                                                                                                                    
                                                                                                                                                                                        @app.get(\"/health\")
                                                                                                                                                                                            def health():
                                                                                                                                                                                                    return {\"status\": \"healthy\", \"model_loaded\": True, \"version\": \"1.0.0\"}
                                                                                                                                                                                                    
                                                                                                                                                                                                        @app.get(\"/model/info\")
                                                                                                                                                                                                            def model_info():
                                                                                                                                                                                                                    return {
                                                                                                                                                                                                                                \"model_type\": type(model).__name__,
                                                                                                                                                                                                                                            \"classes\": CLASS_NAMES,
                                                                                                                                                                                                                                                        \"n_features\": 4,
                                                                                                                                                                                                                                                                    \"feature_names\": [\"sepal_length\", \"sepal_width\", \"petal_length\", \"petal_width\"],
                                                                                                                                                                                                                                                                                \"parameters\": model.get_params()
                                                                                                                                                                                                                                                                                        }
                                                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                                            @app.post(\"/predict\")
                                                                                                                                                                                                                                                                                                def predict(data: IrisInput):
                                                                                                                                                                                                                                                                                                        start = time.time()
                                                                                                                                                                                                                                                                                                                X = np.array([[data.sepal_length, data.sepal_width, data.petal_length, data.petal_width]])
                                                                                                                                                                                                                                                                                                                        pred  = int(model.predict(X)[0])
                                                                                                                                                                                                                                                                                                                                probs = model.predict_proba(X)[0].tolist()
                                                                                                                                                                                                                                                                                                                                        return {
                                                                                                                                                                                                                                                                                                                                                    \"prediction\": pred,
                                                                                                                                                                                                                                                                                                                                                                \"class_name\": CLASS_NAMES[pred],
                                                                                                                                                                                                                                                                                                                                                                            \"probabilities\": probs,
                                                                                                                                                                                                                                                                                                                                                                                        \"inference_time_ms\": round((time.time()-start)*1000, 3)
                                                                                                                                                                                                                                                                                                                                                                                                }
                                                                                                                                                                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                                                                                                                                                                    @app.post(\"/predict/batch\")
                                                                                                                                                                                                                                                                                                                                                                                                        def predict_batch(data: BatchInput):
                                                                                                                                                                                                                                                                                                                                                                                                                results = []
                                                                                                                                                                                                                                                                                                                                                                                                                        for item in data.instances:
                                                                                                                                                                                                                                                                                                                                                                                                                                    X = np.array([[item.sepal_length, item.sepal_width, item.petal_length, item.petal_width]])
                                                                                                                                                                                                                                                                                                                                                                                                                                                pred  = int(model.predict(X)[0])
                                                                                                                                                                                                                                                                                                                                                                                                                                                            probs = model.predict_proba(X)[0].tolist()
                                                                                                                                                                                                                                                                                                                                                                                                                                                                        results.append({\"prediction\": pred, \"class_name\": CLASS_NAMES[pred], \"probabilities\": probs})
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                return {\"predictions\": results, \"count\": len(results)}
